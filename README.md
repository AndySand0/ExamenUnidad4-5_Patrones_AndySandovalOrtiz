Andy Sandoval Ortiz Gestión de Productos Se desarrolló un sistema para gestionar productos donde se permite crear productos, agregar categorías, marcas y etiquetas extras utilizando el patrón Builder para construir productos paso a paso, el patrón Decorator para añadir atributos dinámicos sin modificar la clase base, y el patrón Facade para simplificar la interacción con el sistema de productos. Además, se implementó el patrón Memento para permitir deshacer cambios realizados a los productos y la arquitectura MVC para separar la lógica de negocio, la interfaz de usuario y el modelo de datos, todo mediante un menú interactivo en consola.
